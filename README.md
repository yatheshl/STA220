# STA220 Project: A Comprehensive Analysis of Music Trends

## Commands to setup the environment

1. Create virtualenv in project folder if not created

        python3 -m venv .env

2. Activate virtualenv in project folder

        source .env/bin/activate

3. Install packages from req.txt

        pip install -r req.txt

4. Run jupyter notebook

        jupyter notebook

5. Once done with installing all packages, update req.txt

        pip freeze -l > req.txt

<b> For environment deactivation </b>

        deactivate
